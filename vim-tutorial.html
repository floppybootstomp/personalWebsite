Vim Tutorial For Scrubs (like me)

Here, I'll be making a tutorial on Vim. This tutorial will be for absolute beginners, and only gives the most basic commands to do what you'll need it to. Hopefully this will give you the tools to start editing your files quickly and easily.

Vim is a popular terminal-based text editor for Linux and (recently) Windows. People like it, because it's got more functionality than nano, but it's not bloated like emacs.

How do I install Vim?

  • On Debian-based distros, use sudo apt-get update, then sudo apt-get install vim

How do I use this thing? (The Important Commands)

Launch:

  • Launch vim by typing vim in the terminal.
Open with Vim:
  • You can also open a file with vim by typing vim <filename>.
Modes:
  • In vim, there are two modes. You can't just start typing right away. There is normal mode, where you can type commands, and insert mode where you can freely edit the file like any other text editor.
    • Switch to insert mode by hitting the i key.
    • Switch to normal mode by hitting the Esc key.
CTRL + S:
  • DON'T PRESS IT! Because you're in the terminal, it tells the terminal that the input buffer is full and locks up the program. If you do press it out of habit, use CTRL + Q to unlock it.
Creating and Opening Files:
  • In normal mode, type the command :e <filename>. The : tells Vim you're telling it a command. e is the command to open a file or create one if it doesn't exist.
Saving Your File:
  • :w saves your file. It overwrites the whole file.
Quitting Vim:
  • :q quits the program. :wq saves and quits the program.

Vim Workflow Commands and Techniques

Move Cursor by Word:

  • SHIFT + Left/Right Arrows moves the cursor by word instead of by letter.
Undo:
  • :u is the undo command
Delete Line:
  • :dd deletes an entire line
